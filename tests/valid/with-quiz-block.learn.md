# Python Types

Python is dynamically typed — the type of a variable is determined at runtime.

## Basic types

| Type | Example | Notes |
|------|---------|-------|
| `int` | `42` | Arbitrary precision |
| `float` | `3.14` | IEEE 754 double |
| `str` | `"hello"` | Immutable, Unicode |
| `bool` | `True` | Subclass of int |


## Summary

```summary
- Python infers types at runtime
- str, int, float, bool, tuple are immutable
- list and dict are mutable
```
